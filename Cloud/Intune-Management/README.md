# Intune Management Scripts

This folder contains scripts for managing devices, applications, and configurations through Microsoft Intune.

## Purpose

Scripts in this folder help with:
- Device enrollment and registration
- Application deployment and management
- Configuration profile deployment
- Update policies and management
- Device security settings

## Example Scripts

Place scripts here such as:
- `Enroll-BulkDevices.ps1` - Bulk device enrollment
- `Deploy-Applications.ps1` - Automated app deployment
- `Configure-UpdateRings.ps1` - Windows update ring configuration
- `Get-DeviceInventory.ps1` - Device inventory reporting
- `Set-SecurityBaselines.ps1` - Apply security baseline configurations

## Common Use Cases

1. **Device Enrollment**: Automate device onboarding to Intune
2. **App Deployment**: Deploy line-of-business and store apps
3. **Configuration**: Apply device restrictions and settings
4. **Updates**: Manage Windows updates and feature updates
5. **Security**: Deploy security baselines and compliance policies
